# PNU_INGRO -> [AI 숫자 맞추기 게임 RFP](https://github.com/mrsono0/PNU_INGRO/blob/master/NumericalGame.md "AI 숫자 맞추기 게임 RFP")

지금 까지 배운 내용을 토대로 AI 숫자 맞추기 게임을 만듭니다.
